HTML5 é a quinta e mais recente versão da linguagem de marcação de hipertexto (HTML), usada para estruturar o conteúdo da web. Ele oferece novas funcionalidades e elementos semânticos para aprimorar a experiência do usuário e a eficiência do desenvolvimento web.

CSS3 é a terceira versão da linguagem Cascading Style Sheets (CSS), usada para estilizar páginas web, definindo como elementos HTML são exibidos no navegador.

O QUE É A PROPRIEDADE DISPLAY EM CSS


A propriedade display em CSS determina como um elemento HTML é renderizado na página, controlando seu tipo de exibição (externa e interna) e seu comportamento no layout de fluxo, como em grids e flex boxes. Seus valores mais comuns são block, que faz o elemento ocupar toda a largura disponível e começar numa nova linha, e inline, que faz o elemento ocupar apenas o espaço necessário e permanecer na mesma linha que o conteúdo anterior.

DISPLAY CARACTERISTICAS
BLOCK Em CSS, display: block faz com que um elemento HTML seja renderizado como um bloco, ocupando toda a largura disponível do elemento pai e iniciando em uma nova linha.
INLINE O termo "display: inline" no CSS faz com que um elemento HTML ocupe apenas a largura do seu conteúdo e não gere quebras de linha, permitindo que outros elementos inline apareçam na mesma linha, ao seu lado.
INLINE-BLOCK O display: inline-block em CSS faz com que um elemento seja tratado como um bloco, mas ainda mantenha o comportamento de um elemento inline, permitindo que vários elementos sejam exibidos lado a lado e que propriedades como largura e altura sejam aplicadas.
FLEX O display: flex no CSS transforma um elemento em um contêiner flexível, permitindo que seus filhos (itens flexíveis) sejam dispostos e alinhados de maneira flexível, tanto horizontal quanto verticalmente. É uma ferramenta poderosa para criar layouts responsivos e organizar elementos de forma dinâmica.
